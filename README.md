# Pressing Management System

Web application built with Django for managing a pressing business.

## Features

- Product management
- Order management
- Categories
- Statistics dashboard
- Authentication system

## Tech Stack

- Python
- Django
- SQLite
- HTML / CSS / JavaScript

## Architecture

Built using MVC pattern with database modeling and URL routing.

## Installation

1. Clone the repository
2. Create a virtual environment
3. Install dependencies:
   pip install -r requirements.txt
4. Run migrations:
   python manage.py migrate
5. Start server:
   python manage.py runserver
